#  AdventureWorks Business Intelligence Dashboard

An interactive **Power BI dashboard** developed using the Microsoft AdventureWorks dataset to analyse sales performance, customer behaviour, product profitability, and regional trends. The project demonstrates end-to-end Business Intelligence practices, including data modelling, ETL, DAX calculations, and dashboard design to support data-driven decision-making.

---

#  Project Overview

The objective of this project is to transform raw business data into meaningful insights through interactive dashboards. The report enables stakeholders to monitor key performance indicators (KPIs), identify sales trends, evaluate product performance, and analyse customer purchasing behaviour.

The dashboard was built using a **star schema data model**, Power Query for data transformation, and DAX measures for business calculations.

---

#  Business Objectives

- Monitor overall sales and profitability
- Analyse sales performance over time
- Identify high-performing products and categories
- Evaluate customer purchasing behaviour
- Compare regional sales performance
- Support business decisions through interactive reporting
